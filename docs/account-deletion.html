1. 계정 삭제 방법

위시맵에서는 사용자가 언제든지 계정과 관련 데이터의 삭제를 요청할 수 있습니다. 아래 두 가지 방법 중 하나를 선택하세요.

방법 A — 앱에서 직접 삭제

  1. 위시맵 앱 실행
  2. 하단 [마이] 탭 선택
  3. [설정] → [계정 탈퇴] 메뉴 선택
  4. 안내에 따라 탈퇴 확인

탈퇴 즉시 모든 개인정보가 파기되며, 이 작업은 되돌릴 수 없습니다.

방법 B — 이메일로 요청

앱을 이미 삭제하셨거나 직접 삭제가 어려운 경우, 가입 시 사용한 이메일 주소를 본문에 명시하여 아래 이메일로 삭제 요청을 보내주세요.

  • 받는 사람: cloudwiiiii@gmail.com
  • 제목: [위시맵] 계정 삭제 요청
  • 본문: 가입에 사용한 이메일 주소, 가입 경로(카카오/구글/네이버/Apple)

본인 확인 후 영업일 기준 7일 이내에 처리됩니다.

2. 삭제되는 데이터

계정 삭제 시 아래의 데이터가 즉시 영구 파기됩니다.

  • 계정 정보 (소셜 로그인 식별자, 이메일, 닉네임, 프로필 사진)
  • 작성한 방문 리뷰, 사진, 한줄평, 가격대 정보
  • 작성한 댓글
  • 그룹 멤버십 및 친구 관계
  • 알림 기록
  • 차단/신고 내역
  • 점심 투표 참여 기록
  • 인증 토큰 및 푸시 알림 토큰

3. 보관되는 데이터

아래 데이터는 관련 법령 및 서비스 운영상의 사유로 일정 기간 보관됩니다.

  • 전자상거래법, 통신비밀보호법 등 관련 법령에 따라 보관이 의무화된 정보: 최대 5년
  • 다른 사용자의 콘텐츠 통계에 영향을 주는 비식별·익명화된 집계 데이터: 무기한 (개인 식별 불가능한 형태)

위 보관 기간이 경과하면 해당 데이터도 자동으로 파기됩니다.
