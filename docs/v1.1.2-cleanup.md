## 3. 백엔드 - 하위 호환 엔드포인트 제거 가능 (DEPLOYMENT.md 참조)

| 대상 | 설명 |
|------|------|
| `/restaurants` 경로 alias | `/places`만 남기고 제거 |
| `RestaurantController` 클래스명 | `PlaceController`로 변경 |
| `RestaurantService` 클래스명 | `PlaceService`로 변경 |
